# Personal Finance App (MERN)

Simple personal finance management app built with the MERN stack for learning purposes.

## Features

- User authentication (register, login, JWT, hashed passwords)
- Create and manage multiple accounts
- Add, edit, delete income and expense transactions
- Automatic balance updates
- Monthly budgets per category with status tracking
- Dashboard with total balance, income, expenses, and charts
- REST API (Node.js + Express + MongoDB)
- React frontend with protected routes
